## 🧠 What I'm Learning & Why

### Phase 1 — Python
Coming from JavaScript, Python is the gateway to every AI library.
LangChain, FastAPI, HuggingFace — everything runs on Python.

### Phase 2 — FastAPI + PostgreSQL
Replacing Express + MongoDB with a more powerful AI-ready backend.
FastAPI is async, typed, and production-ready.
PostgreSQL + **pgvector** means I can store both regular data and AI embeddings in one database — no separate vector DB needed.

### Phase 3 — LLM APIs + Prompt Engineering
Learning to communicate with LLMs effectively.
System prompts, structured outputs, tool calling, streaming responses.
This is the core skill every GenAI developer needs.

### Phase 4 — RAG (Retrieval Augmented Generation)
Grounding LLMs in custom data — PDFs, documents, knowledge bases.
Build systems that answer questions from *your* data, not just from model training data.

### Phase 5 — AI Agents
Agents that think, plan, use tools, and act autonomously.
ReAct pattern, LangGraph for stateful flows, multi-step task execution.

### Phase 6 — DevOps & Deployment
CI/CD pipelines with GitHub Actions, containerizing with Docker, deploying on AWS with NGINX.
Because a project that isn't live doesn't exist — and a project that breaks in production isn't ready.
