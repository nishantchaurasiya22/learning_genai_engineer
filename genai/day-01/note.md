# Traditional AI vs Generative AI

```
Traditional AI (umbrella term)
├── Machine Learning (ML)
│   └── Example: Netflix/OTT recommendation
│       (suggests content based on your previous watch history)
│
└── Deep Learning (DL)
    └── Example: Phone face detection/face unlock
        (uses neural networks to recognize faces)

Generative AI (GenAI) - a separate branch
└── Creates NEW content (text, image, video, audio)
    Example: ChatGPT, Midjourney, DALL-E
```

---

# Types of Models inside GenAI

```
GenAI
├── LLM (Large Language Model)
│   └── generates text
│       Example: ChatGPT, Gemini, Claude, LLaMA
│
├── LVM (Large Vision Model)
│   └── generates/understands images and videos
│       Example: DALL-E, Midjourney, Stable Diffusion
│
├── LAM (Large Action Model)
│   └── performs actions, not just content
│       Example: AI agents that complete tasks (booking, clicking, form-filling)
│
├── SLM (Small Language Model)
│   └── lighter and faster version of LLM
│       Example: Phi-3, Gemma, Mistral 7B
│
└── VLM (Vision-Language Model)
    └── understands image + text together
        Example: GPT-4 Vision, Gemini
```

# Quick Reference Table

|-------|-----------|------|---------|

## My Understanding
- LLM sirf text ke liye hai, image ke liye nahi
- Image/video banane wale models alag architecture (Diffusion Model) use karte hain, LLM nahi
- LAM sabse alag hai kyunki yeh content generate nahi karta, balki kaam karta hai (action perform)
- SLM same kaam karta hai LLM jaisa, bas chota aur fast hai (phone/laptop pe chal sakta hai)
- VLM dono samajhta hai — image dikhao aur text mein poochho, wo jawab de dega
